# Number Textures

**10 ready-to-use digit textures (0–9)** for [Stride](https://stride3d.net) — one `Texture` asset
per glyph (512×512, black on white). Handy for prototype UI, debug counters, HUDs, or decals.

## Usage

Open the project in Game Studio — the 10 textures appear under the `Assets` folder, ready to drop
onto sprites, UI images, or materials. Each `Assets/<Digit>.sdtex` sources `Resources/<Digit>.png`.
